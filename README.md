# BIP26_CAN Library
This library provides an interface for the BIP26_CAN module, which is designed to facilitate communication over the CAN bus. The library includes functions for initializing the CAN interface, sending and receiving messages, handling errors, and receiving information about the CAN bus.

## Installation
Simply copy the `BIP26_CAN` folder into the `lib` folder of your PlatformIO project. Then, include the library in your code with:
```cpp#include <BIP26_CAN.h>
```
## Usage
There are two examples provided in the `examples` folder. Refer to the Files for more (specific) information.
