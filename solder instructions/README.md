## Components List
| Identifier | Component | Description | Size |
|------------|-----------|-------------|------|
| `R1`         | Resistor  | 120Ω        | 0603 |
| `R2`, `R3`, `R4`, `R5` | Resistor  | 10kΩ        | 0603 |
| `C1`         | Capacitor | 1µF         | 0603 |
| `C2`, `C3`, `C4`, `C5`| Capacitor | 100nF       | 0603 |
| `IC1`        | Can Transceiver | TJA1051T-3  | SO8 |
| `SW1`       | 4-bit Switch    | NKK DR FC10 | - |
| `J1`        | Pin Header      | 1x18           | 2,54mm |
| `J2`, `J3`    | Pin Socket      | 1x16           | 2,54mm |
| `J4`        | Pin Header      | 1x12            | 2,54mm |
| `J5`        | Pin Header      | 1x02             | 2,54mm |
| `J6`        | DSUB9     | Male            | - |
| `J7`        | DSUB9     | Female          | - | 


## Instructions
I would recommend starting with the SMD components and ICs, and then soldering the connectors and pin sockets/headers. With the IC, correct orientation is important: As shown in the image, it must be aligned so that when you can read the label correctly, the bottom-left pin goes to the position marked with the white arrow. See the red arrows in the image.

![IC_Rotation](https://github.com/BonusPizza/BIP26_CAN/blob/main/solder%20instructions/Pictures/IC_Rotation.jpeg)

For the Sub-D connectors, ensure that the male connector is mounted on the left side of the board and the female connector on the right, as the cable pinout is not symmetrical.
